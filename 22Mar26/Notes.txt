# 2. Introduction to Generative AI (Basic Concepts)

## What is Generative AI?

Generative AI refers to models that can **create new content** such as:

* Text
* Images
* Audio
* Code

Unlike traditional AI (which predicts or classifies), generative AI **produces new data**.

---

## Key Idea

Learn patterns from existing data → generate similar but new outputs.

---

## Types of Generative Models

### a. Large Language Models (LLMs)

* Example: GPT
* Used for:

  * Chatbots
  * Content generation
  * Code assistance

---

### b. Image Generation Models

* Example: DALL·E
* Convert text → images

---

### c. Other Models

* GANs (Generative Adversarial Networks)
* VAEs (Variational Autoencoders)

---

## How It Works (Simplified)

### Step 1: Training

* Model learns patterns from large datasets

### Step 2: Prompt/Input

* User provides input (text, instruction)

### Step 3: Generation

* Model predicts next elements and generates output

---

## Important Concepts

### Tokens

* Smallest units of text (words/subwords)

### Prompt

* Input given to model
* Quality of prompt affects output quality

### Temperature

* Controls randomness:

  * Low → deterministic
  * High → creative

---

## Common Use Cases

* Content writing
* Chatbots / assistants
* Code generation
* Image creation
* Data augmentation

---

## Limitations

* May produce incorrect information (hallucinations)
* Bias from training data
* Requires careful prompt design

---

## Simple Example

Prompt:

> "Write a story about space exploration"

Output:

* AI generates a unique story based on learned patterns
